# Claude Code - דף עזר

---

## מושגים בסיסיים

### פרויקט (Project)
תיקייה שמכילה מסמכים והגדרות עבור הסוכן Claude Code.

### בסיס ידע (Knowledge Base)
מסמכים שנמצאים בתוך התיקייה ומנוהלים על ידי Claude Code.

### Skills
פרומפטים עבור Claude Code שנטענים באופן דינאמי לפי ההקשר.
עוזרים לנו לנהל את חלון ההקשר בצורה אופטימלית.

### Slash Commands
כאשר אנו רוצים לכפות על קלוד פרומפט באופן ידני - גם כשאין הקשר רלוונטי.

### Agents / Subagents
הפעלה של חלון קונטקסט נפרד שמקבל פרומפט ופועל במקביל.
עד שהוא משלים את המשימה - מחזיר את הפלט לשיחה הראשית.

### Hooks
עיתויים (אירועים) במהלך העבודה של Claude Code שניתן להצמיד אליהם הפעלה של סקריפטים.
יכולים לבצע דברים או להתנות את המשך העבודה בבדיקה מסוימת.

> ישנם שני hooks שמאפשרים ביצוע של פרומפט, הקשורים לגמר המענה של קלוד.

### Claude Code Guide
סוכן שמגיע מובנה, שמסוגל לתת מענה מעודכן לגבי היכולות של Claude Code.
נדרש כאשר רוצים להשתמש בפיצ'רים של Claude Code בתוך השיחה.
